# Automated-Linux-Server-Hardening-Security-Audit-Tool
This project is an automated security solution developed for Ubuntu/Linux environments. The primary goal is to simplify the process of server hardening by automating critical security audits, vulnerability scans and permission management. It is designed to minimize the attack surface of a server and ensure that security best practices are followed.
* Key Features
Vulnerability Scanning: Automatically identifies system weaknesses and potential security gaps.
Log Management & Collection: Systematically gathers and organizes system logs (collection_logs.sh) for security auditing.
Automated Hardening: Fixes insecure file and folder permissions within the project environment.
Security Reporting: Generates detailed output reports after each scan for further analysis.
* Repository Structure
/scripts: Contains core Bash scripts for automation.
/logs: Stores captured system security logs.
/output: Contains results and reports from the scanning tools.
* Tech Stack
Operating System: Linux (Ubuntu)
Scripting Language: Bash (Shell Scripting)
Domain: Server Hardening, Vulnerability Assessment.
